decompkernelcache

- Currently implemented: Decompression and extraction of LZVN/LZSS kernel cache

- Changes:

* Added kernel cache kernel extension dict export
* Added kernel extraction
* Added MinGW32 support (both 32 and 64 bit)
* Added Linux support
